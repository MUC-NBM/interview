    # 今年做了什么

    我想，今年做了很多事情，和妈妈去了兰州，和朋友在青海玩，我很喜欢青海，在北京我真的会很想念青海，人总会长大，长意味着人不能做自己喜欢的事情，或者是需要将一些不怎么喜欢的事情变成自己喜欢的事情。 我没见过大海，我想看到蓝色的海，不知道还有没有机会，我想在海里游泳，我想去东帝汶 今年，兰州、泰山、北京，没有认识很多朋友，但是去兰州的那几天过得很开心，或许我喜欢孤独，我的意思是人多很好很热闹，但是那就像奶头乐，独处时我可以真的看见我自己，或者说独处时我会真的很开心，即使是玩也很开心。

    # 兰州

    我挺喜欢兰州的，真的，青海似乎已经没有什么发展前景了，如果青海不是我的故乡，我真的不是很想回去，但是我不想因为这份羁绊就制约住自己，我还想出国，我还想真的做一点事情，我似乎真的是闲不下来，虽然现在十分渴望那种吃喝玩乐的生活，但是倘若真的得到了，我或许还真的会厌恶那种日子。

    如果以后有机会，我真的会想去兰州生活，兰州的普通话也挺普及的，也有地铁，教育环境也不错，交通也不错，气候也不错，不过还要考虑一下四川和西安，可惜都没有兰州离家近。

    # 我是什么样子的人呢？

    没什么打不了的 我一直在问自己？我是什么样的人呢？爸爸说我内向，妈妈说我会说话，但是有时候又会说我不说话，我是什么样子的人呢？有人说我自信，有人说我阴暗，有人说我面瘫，有人觉得我不好说话，有人觉得我软弱可欺。我觉得或许只有我自己才最了解我自己，我爱财，我不爱说话，我没有太多心计，当然，站在我的角度，我总是会说一些褒义词，没有太多心计可以说是无脑。我可能是个自私的人，我不太愿意花时间去了解别人，仔细回想起来是这样的，这些年来也不会经常和朋友联系了，朋友也很少和我联系了，不过还是有几个一辈子的好朋友的，我想这就够了。 如果说还有什么的话。我想我可以是任何人，只是在于开出的条件能不能让我变成那种样子的人。 想起了看过的一则知乎：如果将生活比作一个打怪游戏，我们每个人都是一个角色，无所谓是内向还是外向，在没走向社会之前，内向与外向仅仅是0与的1的区别，也就是说我们都是1级，我把天赋点点到内向上了，而你把天赋点点到外向上了而已。不过感觉自己真的给自己打上了一层内向的标签，其实也就内样，把头发剪了我也很阳光，我可以打篮球，可以剪寸头，可以好好学习，可以满嘴脏话，我可以变成任何样子。以后的路还长着，不过我更想将我的心力专注于我想做的有趣的事情上去。 想起了我在高中那年发了一个qq说说：

    人会看透自己吗？不知道，但是现在的我更爱自己了。

    再放一点说说吧，回忆回忆，这张有个不认识的神经病评论

    # 此刻

    好像什么也写不出来了，突然没什么感觉了，此时此刻正值期末周，还有两天是四六级考试，我的意思是：你会怀念从前吗？ 我想起了多年前穿着那件蓝色冲锋衣的自己，还住在西街，晚上和爸爸妈妈爷爷奶奶去吃火锅，妈妈真的很爱我，我的妈妈是世界上最好的妈妈，我想，我的家庭也还算可以，仔细想想我这一生，在青海出生确实是有点背，但是在青海中上的地区，考到了中上的成绩，不过高考有一说一确实还可以，不过没有条件复读，不然我想我不会止步于此。高考全省1700名，少数民族排名310,在小县城能考到这种成绩，应该还算可以吧，小小骄傲一下，嘻嘻。中上的家庭，中上的成绩，中上的身高，中上的颜值，中上的身材，但是我总觉得我的这种心境，或许这样说来有些狂，总觉得自己的心态很成熟，有种超越同龄人的成熟。

    或许只是我自己的一厢情愿吧。

    不过我希望，你可以认真考虑自己要不要留学，你可以想想自己要考公还是工作，中国现在的社会体制真的很怪，如果教育体制不改革，后果会是什么样子呢？

    如果你想考研，我希望你可以好好规划自己的人生，要不要出国工作，要考哪个学校，要不要留在北京。这些事情还是都得考虑考虑的。

    好想抱抱以前的自己，也好像抱抱现在的自己。希望我可以考上我想考的学校，可以有很大的成就，可以快乐地度过我接下来的人生。
