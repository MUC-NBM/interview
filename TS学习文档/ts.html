    # 4. 联合类型

    联合类型使用 | 分隔每个类型：

    let myFavoriteNumber: string | number;
myFavoriteNumber = 'seven';
myFavoriteNumber = 7;

    # 编译脚本

    tsc app.ts

    也可以一次性编译多个脚本文件

    tsc file1.ts file2.ts file3.ts

    编译到指定目录，使用指定的es版本编译

    # 编译报错

    编译报错产生产物或者只针对类型检查

    # tsconfig.json

    {
  "files": ["file1.ts", "file2.ts"],
  "compilerOptions": {
    "outFile": "dist/app.js"
  }
